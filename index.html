Local Dimensions

A little plugin to show window dimensions during local development.

Re-size your browser to see it in action!

Call it like so:

$('body').localDimensions({devUrl: "your_local_server_url"});

Why pass the local server url?
This is simply so you don't have to disable the plugin everytime you publish to a staging URL during the development phase. When you're done with development, you can remove the plugin altogether.

Hint: Your local server url needs to match what's in your url location bar in the browser. Find your exact local url by checking the value of window.location
